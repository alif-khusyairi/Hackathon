# 🚀 Hackathon Project – Smart Clinic Workflow System

## 📌 Overview

The **Smart Clinic Workflow System** is a web-based application designed to improve efficiency, organization, and patient flow in clinics.

This system helps clinics manage:

* Patient queue
* Appointment scheduling
* Staff workflow
* Real-time dashboard monitoring

Our goal is to reduce waiting time, improve communication, and create a smoother healthcare experience for both patients and staff.

---

## 🎯 Problem Statement

Many clinics still rely on manual systems or inefficient digital tools, which lead to:

* Long waiting times
* Poor queue management
* Lack of real-time updates
* Inefficient staff coordination

This project solves these issues by providing a centralized and intelligent workflow system.

---

## 💡 Solution

We developed a **Smart Clinic Workflow System** that:

* Digitizes clinic operations
* Provides real-time tracking of patients
* Improves staff coordination
* Enhances user experience with a clean and intuitive interface

---

## ⚙️ Features

### 🧑‍⚕️ Patient Management

* Register and manage patient records
* Track patient status (waiting, in consultation, completed)

### 📊 Dashboard System

* Real-time overview of clinic activities
* Monitor queue and workflow status

### 📅 Appointment Scheduling

* Book and manage appointments
* Reduce overcrowding and waiting time

### 🔄 Workflow Automation

* Automatically update patient progress
* Optimize staff task distribution

### 🎨 User-Friendly Interface

* Clean UI using modern web technologies
* Easy navigation for staff and users

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/alif-khusyairi/Hackathon.git
```

### 2️⃣ Navigate to the project folder

```bash
cd Hackathon
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run dev
```
